Metropolis rule: propose a single spin flip; P(flip) = min(1, e−ΔE/kBT) is the probability of accepting the flip. If rejected (probability 1 − P(flip)), the system stays in its current state. Downhill moves (ΔE ≤ 0) are always accepted.

Keyboard shortcuts: Space pick random site / let Metropolis decide   F force flip   K keep   N new random state

Ising energy: E = −J Σ⟨ij⟩ sisj (nearest-neighbour, periodic boundary, J = 1). The 2D critical temperature is kBTc / J ≈ 2.269.
